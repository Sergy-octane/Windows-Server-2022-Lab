# 🚀 Windows Server 2022 & Active Directory Lab

This project documents the end-to-end implementation of a virtualized enterprise network environment.

## 🛠️ Tech Stack
* **Hypervisor:** VirtualBox
* **Server OS:** Windows Server 2022 (Domain Controller)
* **Client OS:** Windows 10 Pro
* **Networking:** Static IP configuration, DNS resolution, and Bridged Adapter setup.
